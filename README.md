## شرح آزمایش 
ابتدا مانند آزمایش های قبلی مخزن گیت پروژه را راه اندازی کرده و تنظیمات محافظت از برنچ main را روی آن اعمال می کنیم. 

### بخش اول :‌ بررسی پوشش آزمون در json-simple
